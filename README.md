# 🚁 Drone Detection System

A fast, real-time drone detection web dashboard. Built with Python, Streamlit, YOLO, and OpenCV, it tracks drones in live webcam feeds, images, and videos with zero lag.

## ✨ Features
* **Live Webcam Tracking:** Connect your built-in or external camera to detect and track drones in real-time.
* **Photo & Video Analysis:** Easily drag and drop `.jpg`, `.png`, `.mp4`, or `.mov` files to see instant detections.
* **Smooth Video Playback:** Uses background H.264 video rendering to ensure processed videos play back smoothly in the browser without freezing.
* **User-Friendly Dashboard:** A clean, modern web interface that makes running AI models incredibly simple.

## 🛠️ Tech Stack
* **Language:** Python
* **Frontend:** Streamlit
* **Computer Vision:** OpenCV (`cv2`)
* **Machine Learning:** Ultralytics (YOLO)
* **Data Processing:** NumPy, Pillow
